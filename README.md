# React Native com RealDB

- Configuração inical do projeto executar:

```jsx
npm install
ou
yarn
```

- Para executar projeto executar:

```jsx
yarn android
```

### Projeto realizado para estudo da plataforma com o banco de dados RealmDB.

Documentação utilizada: 

[https://reactnavigation.org/docs/getting-started](https://reactnavigation.org/docs/getting-started)

[https://docs.mongodb.com/realm/sdk/react-native/quick-start-local/](https://docs.mongodb.com/realm/sdk/react-native/quick-start-local/)
